H-Hyper : Being Super Interactive (HTML is very interactive as it can be executed in any browser, and can get the data from anywhere in the web)
    Some features of HTML which make it hyper
  • Anchor tag "a"
  • Forms

T-text: In HTML text means any visible component in a webpage. These reside in the body of the page. Text can be plain text, images, video, links, tables etc.
M-Markup : In HTML the tags and styling are major part that describe the metadata of the page. Markup in plain english means formatting. We do formatting in a webpage with the help of tags and CSS
L-Language : All Computer languages have common features of having:
  • Variables: placeholders to actual value
  • Statements: lines of codes, like sentences
  • Control Structures: if....else if...else, switch case
  • Loops: repeatative structures
  • Functions: group of statements solving a particular problem
In HTML we have Javascript handling the language part.

CSS: Cascading: following the HTML file. Style: Styke the HTML content. Sheet: Place where we wrtie the style code.

JS: JavaScript is the language behind HTML

TS: TypeScript is the new language which is strongly typed. TS gets compiled into JS and then executed in HTML.
